# Central Studio
## 概述
&emsp;&emsp;Central Studio 是一个强大的微服务开发套件，包含了一系列常用的、通用的基础服务。目前包括：

- 网关中心(3000)
- 认证中心(3200)
- 数据服务中心(3300)
- 日志中心(3400)
- 存储中心(3500)
- 广播中心(3600)

&emsp;&emsp;Central Studio 在接下来会继续完善和丰富。

&emsp;&emsp;访问官网[[链接](https://central-x.com)]获取更多文档信息。
